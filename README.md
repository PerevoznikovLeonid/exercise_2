# .ЗАДАНИЕ 2.

Создать программу по конвертации данных из одного формата в другой. Использовать чтение и запись в файл, работать с пользователем через меню, использовать структуры.

### Исходный формат данных

Данные могут разделяться разными символами. Например:
```psv
LastName|FirstName|00.00.0000
LastName|FirstName|00.00.0000
```

# Результирующий формат данных
```json
[
  {
    "last_name": "LastName",
    "first_name": "FirstName",
    "date_of_birth": {
      "year": 0000,
      "month": 00,
      "day": 00
    }
  },
  {
    "last_name": "LastName",
    "first_name": "FirstName",
    "date_of_birth": {
      "year": 0000,
      "month": 00,
      "day": 00
    }
  }
]
```
